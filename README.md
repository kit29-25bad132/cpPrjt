# cpPrjt
C project given in college.

HERE'S WHAT I DO IN THIS PROJECT:

Write a C program using structures, functions, and switch-case with the following requirements:

Create a structure Student to store name, roll number, and marks.

Read the details of 3 students and store them in an array of structures.

Write a function to find and return the index of the student with the least marks.

Write another function to print the details of a student.

Use a switch-case menu with options to:

Find the student with the least marks

Print details of all students

Exit
